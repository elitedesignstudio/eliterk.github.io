In computing, an icon is a pictogram or ideogram displayed on a computer screen in order to help the user navigate a computer system. The icon itself is a quickly comprehensible symbol of a tool, function, or a data file, accessible on the system and is more like a traffic sign than a detailed illustration of the actual entity
it represents.
